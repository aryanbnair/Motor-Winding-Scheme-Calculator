# Motor Winding Scheme Calculator

A single-file HTML tool for generating and evaluating three-phase winding schemes for electric machine stators. It calculates winding factors, supports custom schemas, visualizes slot/phase allocation, and renders a stator diagram directly in the browser.

## Features

- Calculate winding schemes from slot and pole combinations
- Support for custom winding schema input
- Turn-by-turn weighting through editable turn inputs
- Winding factor calculation for the selected configuration
- Harmonic winding factor table generation
- Optional slot factor correction
- Optional skew correction
- Stator rendering / visual winding layout
- Theme toggle
- English and German (Incomplete) UI support

- ## How it works

The calculator follows this general flow:

1. Read user inputs such as slots, poles, layer type, shortening, connection type, and phase naming.
2. Validate the input combination.
3. Build or accept a winding schema string.
4. Convert the schema into editable turn inputs.
5. Calculate the winding factor from the schema.
6. Generate a harmonic winding factor table.
7. Apply optional slot-factor or skew-factor corrections.
8. Render the stator visualization.

## Core concepts

### 1. Winding schema

The application uses a schema string as its core internal representation of the winding.  
Phase letters represent assigned coil sides, while separators and placeholders encode slot relationships and empty positions.

### 2. Turn weighting

Each active schema element can receive its own turn value.  
These values are read from dynamically generated input fields and passed into the winding-factor engine, allowing non-uniform coil turns.

### 3. Winding factor calculation

The main calculation logic is handled by `calcWF(...)`.  
It evaluates the schema against slot count, pole count, and turn weights.

The calculator can detect when a scheme is unbalanced and returns separate values when needed.

### 4. Harmonic analysis

The harmonic table is generated by `generateWFTable(...)`.  
This allows the user to inspect winding-factor behavior over multiple harmonic orders instead of only the selected operating point.

### 5. Correction factors

Optional correction terms can be enabled for:

- slot factor
- skewed slots

These depend on additional geometric inputs such as stator diameter, slot opening, and skew slot count.

## Usage

Open the HTML file in a browser and then enter the machine parameter
